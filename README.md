### 🔒 My latest project

**HitchHiker** - Linux covert communication platform via  Net steganography.  
Messages are encrypted, fragmented, and embedded into unused fields of existing network traffic. No new packets generated. Data rides what's already moving. Undetectable by design.

`C++20` `Redis` `SQLite` `Crypto++` `Docker` `CMake` `Linux` `AWS EC2`
