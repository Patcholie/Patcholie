## 〉Featured Project

### [HitchHiker](https://patcholie.github.io) — Network Steganography Platform
`C++20` · `2025–2026` · `Completed` · *Co-developed with [Daniel Varkin](https://github.com/marmust)*

A production platform for covert communication via pure network steganography. Messages are **encrypted, fragmented, and embedded into unused fields of live network traffic**. No new packets generated. Data rides what's already moving. **Undetectable by design.**

| Component | Description |
|---|---|
| `hitchhiker_lib` | Core: cryptography, fragmentation, transport protocol, steganographic embedding |
| `hitchhiker_server` | Headless: kernel packet interception, storage (Redis + SQLite), routing |
| `hitchhiker_client` | Full TUI via FTXUI — coordinator logic, connection monitoring |

- **HitchHikerAPI** — unified interface; neither client nor server knows the underlying protocol mechanics
- **Pluggable carrier system** — embedding targets fully modular; adding or removing a carrier changes nothing else
- **Adaptive embedding** — multiple fields across multiple protocols per packet; bit-field precision throughout
- **Full crypto pipeline** — ECDH secp256r1 · ChaCha20 · HKDF-SHA256 · Argon2id · CRC-32 via Crypto++

`C++20` `CMake` `Crypto++` `libnetfilter_queue` `iptables` `FTXUI v5` `Redis` `SQLiteCpp` `spdlog` `tomlplusplus` `Docker` `OpenVPN` `AWS EC2`
